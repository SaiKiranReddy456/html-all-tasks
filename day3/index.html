Visual Code Setup

Visual Studio Code (VS Code) is a lightweight but powerful source code editor developed by Microsoft. It is a streamlined code editor with support for development operations like debugging, task running, and version control. It aims to provide just the tools a developer needs for a quick code-build-debug cycle and leaves more complex workflows to fuller featured IDEs, such as Visual Studio IDE.

Features of VScode:

  • Syntax Highlighting and Auto-Completion for most languages.
  • Integrated Terminal for running scripts or commands.
  • Version Control Integration for Git and other systems.
  • Extensions Marketplace to add functionality, such as language-specific support, linters, debuggers, themes, and more.
  • Customizable Interfacewith user-defined settings and shortcuts.
  • Built-in Debuggingfor a seamless coding experience.

    • Emmet abbreviations:

    Emmet Symbol Description
    ! Expands to the complete html5 template.
    tagname Expand the complete the tag.
    > Used to create child elements within a parent element.
    + Used to create sibling elements.
    ^ Moves up one level in the hierarchy, allowing you to add sibling elements to parent elements.
    * Create multiple elements.
    {} For content.
    [] For attributes.

    Shortcuts:

    Shortcut Description
    Ctrl+S Save the current file.
    Ctrl+O Opens the selected file.
    Ctrl+C Copy the selected text.
    Ctrl+V Paste the selected text.
    Ctrl+X Cut.
    Ctrl+Z Cut the selected text or item.
    Ctrl+Y Redo the previously undone action.
    Ctrl+A Select all content in the current file.
    Ctrl + Shift + P Open Command Palette.
    Ctrl + Shift + N New window.
    Ctrl + Shift + W Close window.
    Ctrl + Shift + Alt + Down Add cursors to multiple lines for simultaneous editing.
    Alt + Up Arrow Move the selected line or block of code one line up.
    Alt + Down Arrow Move the selected line or block of code one line down.
